#MAZE PROJECT

## How to Install
To install, you need access to a Terminal
### On Linux
On a Linux system, hold `ctrl + alt + t` to pop up a terminal.
-	git clone https://github.com/0xElectrifier/MAZE\_Project/
-	cd MAZE\_Project/
-	./build\_game

On a Windows system, hold XXXXX to pop up a terminal.
-	git clone
